## Deskripsi Project
Project ini berisi implementasi cloud storage menggunakan Ubuntu Server/Desktop dengan Apache, MySQL, PHP, dan Nextcloud/ownCloud.

## Tujuan
- Memahami konsep cloud storage
- Implementasi server berbasis web
- Konfigurasi database dan web server

## Teknologi yang digunakan
- Ubuntu 24.04
- Apache2
- MySQL
- PHP
- Nextcloud (compatible with PHP 8.3)

## Langkah Instalasi Singkat
1. Install Ubuntu
2. Install Apache, MySQL, PHP
3. Download dan setup cloud storage
4. Konfigurasi database
5. Akses melalui browser
